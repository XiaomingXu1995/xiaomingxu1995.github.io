# toolkit for bioinformatics

## online phylogenetic tree (newick tree) viewer online
ETEToolkit 线上根据newick tree 格式进行自动viewer的网站: [`newick viewer`](http://etetoolkit.org/treeview/)

NCBI Tree view 有关newick tree format的预览网站: [`NCBI newick viewer`](https://www.ncbi.nlm.nih.gov/projects/treeview/)
